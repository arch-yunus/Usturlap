# Contributing to Usturlap

Welcome! We are excited that you want to contribute to the **Usturlap (Astrolabe)** project. This project aims to be the gold standard for astrological calculations.

## Code of Conduct
By participating in this project, you agree to maintain a professional and respectful environment.

## How to Contribute
1.  **Fork the Repository:** Create your own copy of the project.
2.  **Create a Branch:** `git checkout -b feature/amazing-feature`
3.  **Implement Your Changes:** Ensure your code follows the established style and includes tests.
4.  **Run Tests:** `pytest tests/`
5.  **Submit a Pull Request:** Describe your changes in detail in the PR template.

## Technical Standards
-   **Strict Typing:** Always use Pydantic models for data structures and type hints for functions.
-   **Precision:** Use Swiss Ephemeris (`pyswisseph`) for all calculations. Avoid re-implementing basic astronomical math.
-   **Documentation:** Update the `README.md` if you add new features.

## Professionalism
Maintain a high-density, technical, and scientifically neutral tone in all documentation.
